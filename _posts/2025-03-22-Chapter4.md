## 스택

### 스택이란, 스택 설명추가

스택 이미지 추가

```c
#include <stdio.h>

int stack[5];
int top = -1;

int isEmpty(){
    return (top == -1);
}

int isFull(){
    return (top == 4);
}

void push(int value){

    if (isFull() > 0)
    {
        return; // 꽉 찼으면 못 넣는다.
    }

    top++;
    stack[top] = value;
}

int pop(){

    if (isEmpty() > 0)
    {
        return -999;    // 비어있다는 의미로 -999 반환
    }

    int result = stack[top];
    top--;

    return result;
}

int main(){

    return;
}
```
